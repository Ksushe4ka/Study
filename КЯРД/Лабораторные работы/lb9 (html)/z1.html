Что такое HTML
HTML (HyperText Markup Language) представляет язык разметки гипертекста, используемый преимущественно для создания документов в сети интернет
Элементы являются кирпичиками, из которых складывается документ html5.
Стили определяют представление документа, его внешний вид. Определение стиля состоит из двух частей:
CSS3
селектор
блок объявления стиля
HTML5 и CSS3
